This is the code used for search the degree-4 sbox in the paper.

Note that the degree-4 sbox in the paper is actually the equivalent of known results in [Ras23] (Low-Latency Boolean Functions and Bijective S-boxes. (2022). IACR Transactions on Symmetric Cryptology, 2022(3), 403-447. https://doi.org/10.46586/tosc.v2022.i3.403-447)

## Usage of the codes 

python Sbox_Search.py



## Results
It will return the S-box that meets the requirements, enter any command to search for the next S-box that meets the requirements
